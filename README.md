## Требования к разработанному ПО
Реализовать лабораторную работу по предмету "Вычислительная математика" посвящённую интерполяции со следующими дополнениями:
- обязательно должна быть реализована линейная интерполяция;
- настройки алгоритма интерполяции и выводимых данных должны задаваться через аргументы командной строки;
- входные данные должны задаваться в текстовом формате на подобии ".csv" (к примеру x;y\n или x\ty\n) и подаваться на стандартный ввод, входные данные должны быть отсортированы по возрастанию x;
- выходные данные должны подаваться на стандартный вывод;
- программа должна работать в потоковом режиме (пример -- cat | grep 11), это значит, что при запуске программы она должна ожидать получения данных на стандартный ввод, и, по мере получения достаточного количества данных, должна выводить рассчитанные точки в стандартный вывод.

## Ключевые моменты реализации

### Алгоритмы интерполяции
1. **Линейная интерполяция** (`--linear`) - интерполирует между двумя соседними точками.
2. **Интерполяция Лагранжа** (`--lagrange n`) - интерполирует с использованием n точек.

### Параметры командной строки
- `--step s` - задает шаг дискретизации (по умолчанию 0.1)
- Можно использовать несколько алгоритмов одновременно

### Архитектура
Программа состоит из трех модулей:
1. `Interpolations` - реализация алгоритмов интерполяции.
2. `Io` - функции ввода/вывода.
3. `Main` - основная программа с обработкой аргументов командной строки.

## Использование
```bash
# Линейная интерполяция
echo -e "0 0\n1 1\n2 2" | dune exec fp-lab3 -- --linear

# Интерполяция Лагранжа с 3 точками
echo -e "0 0\n1 1\n2 2\n3 3" | dune exec fp-lab3 -- --lagrange 3

# Линейная интерполяция с шагом 0.5
echo -e "0 0\n1 1\n2 2" | dune exec fp-lab3 -- --linear --step 0.5

# Интерполяция Лагранжа с 3 точками и шагом 0.2
echo -e "0 0\n1 1\n2 2\n3 3" | dune exec fp-lab3 -- --lagrange 3 --step 0.2

# Использование нескольких алгоритмов одновременно
echo -e "0 0\n1 1\n2 2" | dune exec fp-lab3 -- --linear --lagrange 3
echo -e "0 0\n1 1\n2 2\n3 3" | dune exec fp-lab3 -- --linear --lagrange 4 --step 0.5
```
